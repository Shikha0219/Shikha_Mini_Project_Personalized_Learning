#  Personalized Learning Path Generator
This project is a Machine Learning-based web application that predicts student performance and generates a personalized learning path.
The system takes student input data, predicts performance score using a trained ML model, and provides recommendations like Beginner, Intermediate, or Advanced learning paths.

##  Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn
- Matplotlib
- MongoDB Atlas (Cloud Database)

---

##  Features
✔ Manual student input via UI  
✔ ML-based performance prediction  
✔ Personalized learning recommendations  
✔ Data storage in cloud (MongoDB)  
✔ Performance visualization (graph)  
✔ Fully deployed online  

---

##  Input Parameters
- Hours Studied  
- Previous Scores  
- Extracurricular Activities  
- Sleep Hours  
- Sample Question Papers Practiced  

---

##  Machine Learning Models Used
- Linear Regression (Selected Model)
- Random Forest
- Gradient Boosting

 **Why Linear Regression?**  
It provided the best performance based on:
- Lowest Mean Squared Error (MSE)
- Highest R² Score

---

## Output
- Predicted Performance Score  
- Learning Path Recommendation:
  - Beginner
  - Intermediate
  - Advanced
